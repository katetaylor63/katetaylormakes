Layout

Helper Classes

There are a series of classes built that will provide padding and margin values for all sides, individual, or pairings. The values are derived from calculations based on a single variable - that way the site's gutters can be scaled up or down easily and not break the layout.

Variables
$base: (6)
$xs: $base * 1 + px
$sm: $base * 2 + px
$md: $base * 3 + px
$lg: $base * 4 + px
$xl: $base * 10 + px
$xxl: $base * 20 + px
